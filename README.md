# QR Code Generator

Sebuah aplikasi web sederhana dan cepat untuk membuat QR Code, dibangun menggunakan Python (Flask) dan antarmuka modern yang menarik.

## 🌟 Fitur

- **Pembuatan Instan**: Ubah URL atau link apa pun menjadi QR Code secara instan.
- **Kustomisasi Warna**: Sesuaikan warna QR Code dan warna latar belakang sesuai selera Anda.
- **Download & Copy**: Unduh QR Code sebagai file PNG atau salin langsung ke clipboard.
- **Desain Responsif**: Antarmuka pengguna (UI) modern dengan gaya *glassmorphism* yang tampil optimal di perangkat desktop maupun seluler.

## 🛠️ Teknologi yang Digunakan

- **Backend**: Python, Flask, `qrcode`, `Pillow`
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)

## 🚀 Cara Menjalankan Project

1. **Clone repositori ini** (atau ekstrak file zip project):
   ```bash
   git clone https://github.com/username-anda/QRcreat.git
   cd QRcreat
   ```

2. **Buat Virtual Environment (Opsional namun disarankan)**:
   ```bash
   python -m venv venv
   
   # Untuk Windows:
   venv\Scripts\activate
   
   # Untuk Linux/Mac:
   source venv/bin/activate
   ```

3. **Instal dependensi**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Jalankan aplikasi**:
   ```bash
   python app.py
   ```

5. **Buka di Browser**:
   Buka web browser Anda dan akses `http://127.0.0.1:5000`
